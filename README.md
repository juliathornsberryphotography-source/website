# Simple Photography Website

This folder contains a simple one-page website ready to publish on Cloudflare Pages.

## What to change
- Replace the placeholder image in images/header-photo.svg with your own header photo.
- Replace the placeholder image in images/price-list.svg with your pricing image.
- Update the Instagram link and email address in index.html.

## Publish on Cloudflare Pages
1. Zip this folder or connect it to a GitHub repository.
2. In Cloudflare Pages, create a new project and upload the folder or connect your repo.
3. Set the build output directory to the root folder if needed.
4. Publish.
